Linux System Administration & Shell Scripting Research
This repository serves as a comprehensive technical guide and resource hub for Linux User Management, Package Administration, and Shell Logic. Developed as part of a research quiz presentation, the project explores the foundational pillars of Linux environments, focusing on the critical distinctions between standard users and the Root/SuperUser through sudo implementation.

Key areas covered include:

User Lifecycle Management: Understanding UID 0, access control, and account permissions.

Package Ecosystems: Practical workflows for Debian-based systems (apt), Arch-based systems (pacman), and universal script-based installations.

Shell Architecture: Deep dives into shell types and the execution of complex command-line logic.

Advanced Data Filtering: Mastering the grep command using AND/OR/NOT logic and recursive searching to manage system logs and configuration files efficiently.
